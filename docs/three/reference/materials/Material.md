# Material

マテリアルの抽象的な基底クラス．

マテリアルは，オブジェクトの外観を記述する．レンダラに依存しない方法で定義されているため，別のレンダラを使用することになってもマテリアルを書き直す必要はない．

以下のプロパティとメソッドは，他のすべてのマテリアルタイプに継承される（ただし，デフォルトは異なる場合がある）．

## コンストラクタ

```js
Material();
```

これにより，汎用的な素材ができあがる．
